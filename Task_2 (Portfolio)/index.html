 Project-2: "Payment Gateway Integration"

        
                    
Description:
            I have  created a simple website where payment gateway is integrated.
                    On the homepage, a basic "Donate" button for donation
                    should be present. The user can choose the amount to 
                    donate and the method of payment (credit card, Paypal, etc.)
                    on the payment page after clicking the donate button.

                    email will be sent to the user for the payment received.

Technologies Used:

  • HTML
  • CSS
  • Java-Script programming language
  • VS code tool
  • Payment gateway API

Key Features:

  • Secure payment processing
  • User-friendly payment form
  • Support for multiple payment methods
  • Transaction management and reporting

(i)Security and compliance: One of the biggest
                challenges when creating this website is ensuring that the
